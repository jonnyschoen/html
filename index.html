How do you add color?

Color is easy to add. Unfortunately, you can't add it directly from the HTML library. Instead, you need to learn a seperate coding language call CSS. Using CSS you can add color and customization to your web page! There are many colors you can add such as red, blue, green, teal, brown, and much, MUCH more!

Lists

Lists give the user short, quick, bulleted information. This can be great if you're trying to add some space between the list and the paragraph. This is because reading a huge bulk of information can be hard and boring. So splitting it up a little really helps!
There are two types of lists: organized lists and unorganized lists. Here is an example:

  1. This is an organized list
  2. This is also an organized list
  3. Organized lists are numbered

As you can see, organized lists are numbered. This is why they are called organized. Here is an unorganized list:

  • This is an unorganized list
  • This is also an unorganized list
  • Unorganized lists use bullet points instead of numbers

This is called an unorganized list. This is because it uses bullets instead of numbers like an ordered list. This is great short information that the user doesn't really need to keep track of.
