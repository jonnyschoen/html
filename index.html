Color is easy to add. Unfortunately, you can't add it directly from the HTML library. Instead, you need to learn a seperate coding language call CSS. Using CSS you can add color and customization to your web page! There are many colors you can add such as red, blue, green, teal, brown, and much, MUCH more!
